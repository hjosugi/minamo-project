# Review Agent Prompt

> 日本語版: [review-agent-prompt-ja.md](review-agent-prompt-ja.md)

Review a KGM1 PR.

Key review points:

- Does jitter increase?
- Do fingers or joints break?
- Do the eyes or mouth tremble unnaturally?
- Do drum hit false positives increase?
- Is raw video ever transmitted?
- Are the issue's acceptance criteria met?
- Are there tests?
- Is there a fallback?

Output:

- Blocker
- Should fix
- Nice to have
- Good points
- Suggested patch
