# Review Agent Prompt

> English version: [review-agent-prompt.md](review-agent-prompt.md)

KGM1のPRをレビューしてください。

重点レビュー項目:

- ガタつきが増えないか
- 指や関節が折れないか
- 目や口が不自然に震えないか
- ドラムhit false positiveが増えないか
- raw videoが送信されないか
- Issueの完了条件を満たすか
- テストがあるか
- fallbackがあるか

出力:

- Blocker
- Should fix
- Nice to have
- Good points
- Suggested patch
