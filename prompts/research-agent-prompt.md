# Research Agent Prompt

> 日本語版: [research-agent-prompt-ja.md](research-agent-prompt-ja.md)

Research targets:

- MediaPipe Tasks
- ONNX Runtime Web
- WebGPU

Output format:

1. Conclusion

3. Reasons to adopt

5. MVP implementation plan
6. Risks
7. Issue candidates
