KAGAMI

普通のwebcamひとつで、誰でも無料でアバター配信のトラッキングを。推論はすべてブラウザ内。ネットワークに出るのは1フレーム約76バイトの動きデータだけ。

signal in

Tracker

webcam → 52ch表情 + 頭部姿勢。One Euroフィルタで平滑化してKGM1バイナリで送出。

 signal out

Viewer

KGM1を受信してVRMアバター(または内蔵ボット)を駆動。OBSのブラウザソースにそのまま載る。

 spec

Docs

KGM1プロトコル仕様、アーキテクチャ、ロードマップ。docs/ ディレクトリ参照。
quick start

最速の試し方(サーバ不要)

このサイトを静的配信(scripts/dev.sh か npx serve)→ Trackerで「Start tracking」→ localモードでConnect → 「ビューアを別タブで開く」。同一ブラウザ内でBroadcastChannel経由で動きます。
