Minamo

Avatar-streaming tracking for everyone, free, with just an ordinary webcam. All inference runs in the browser. The only thing that leaves your machine is about 76 bytes of motion data per frame.

signal in

Tracker

Webcam → 52-channel facial expression + head pose. Smoothed with a One Euro filter and sent as KGM1 binary.

 signal out

Viewer

Receives KGM1 and drives a VRM avatar (or the built-in bot). Drops straight into an OBS browser source.

 spec

Docs

KGM1 protocol spec, architecture, and roadmap. See the docs/ directory.
quick start

Fastest way to try it (no server needed)

Serve this site statically (scripts/dev.sh or npx serve) → click "Start tracking" in the Tracker → Connect in local mode → "Open viewer in a new tab". It works within the same browser via BroadcastChannel.
