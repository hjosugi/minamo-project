Local-first avatar tracking

普通のWebカメラから、
自然なアバターの動きへ。

Minamoは、顔・目・口・手・指・上半身、そしてドラム演奏までを扱う オープンなトラッキング基盤です。映像は端末内で処理し、外へ送るのは小さなモーションデータだけです。

30秒デモを試すカメラなしでも動作 実トラッカーを開くカメラ許可が必要
  • LOCAL 映像をアップロードしない
  • OPEN 0BSDライセンス
  • WEB インストール不要のデモ

What it is

カメラ映像を、配信で使える動きに変える。

01

Webカメラで検出

顔の表情、頭の向き、手や指のランドマークをブラウザ内で取得します。

→
02

破綻しにくく安定化

フィルタ、速度制限、解剖学的制約、遮蔽復帰でガタつきや不自然な曲がりを抑えます。

→
03

好きなアバターへ

VRM、実験的なInochi2D、レイヤー画像、OBS向け表示へモーションを渡します。

Try it now

まずは、いちばん簡単な入口から。

データのアップロードやアカウント作成はありません。用途に合わせて3つの入口を選べます。

おすすめ UIデモ

カメラなしでも30秒で試す

顔・手・ドラム信号のモック可視化を起動します。カメラを許可すると背景に重ねて確認できます。

デモを開く → 実データ Tracker

実際のWebカメラで動かす

トラッカーを開始し、localモードで接続。別タブのViewerへ同一ブラウザ内で送信します。

Trackerを開く → 出力 Viewer

アバター表示を確認する

内蔵ボットから始め、VRM / GLB / Inochi2Dファイルや記録済みJSONLをローカルで読み込めます。

Viewerを開く →

デモについて: UIデモの骨格と数値はシミュレーションです。実際のMediaPipeトラッキングを試す場合はTrackerを使用してください。
