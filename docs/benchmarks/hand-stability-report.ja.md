# 手の安定性ベンチマークレポート

日付: 2026-07-06

範囲: オープンパーム、ポイント、拳、不可能なカールジャンプ、短い遮蔽回復、長い遮蔽省略のための合成回帰カバレッジ。

## アセット

- Fixture: `tests/fixtures/hand-golden-clip.json`
- ビジュアルページ: `diagnostics/no-broken-finger.html`
- ランタイム: `HandTargetStabilizer`, `classifyHandGesture`,
  `handTargetDebugRows`

## ゲート

| ゲート | 結果 |
|---|---|
| 16バイトの手のターゲットがデコード可能である | `tests/run-tests.mjs` によってカバーされています |
| 不可能なカールジャンプがクランプ警告を発する | ゴールデンクリップテストによってカバーされています |
| フレームごとのカールステップが <= 0.24 に安定している | ゴールデンクリップテストによってカバーされています |
| 短い手の不在が回復フラグで保持される | ランタイムテストによってカバーされています |
| 長い手の不在が手のブロックを省略する | ランタイムテストによってカバーされています |
| ビューアにアームソルバーのフォールバックトグルがある | 構造検証によってカバーされています |

## 手動使用

開発サーバーを実行し、`/diagnostics/no-broken-finger.html` を開きます。チャートは合成クリップに対する安定化された人差し指のカールを示します。黄色のポイントは回復保持出力を示し、警告行はクランプおよび回復イベントをリストします。
