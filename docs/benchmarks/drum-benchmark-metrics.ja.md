# ドラムベンチマークメトリクス

ステータス: イシュー #121 と #123 のベンチマーク定義が実装されました。

イシュー #234 のローカルランナーは、[drum-benchmark-runner.md](drum-benchmark-runner.md) に文書化されています。

## クリップセット

最小ローカルクリップセット:

- 60 bpm でのシングルスネアヒット
- 120 bpm での交互の左/右スネアヒット
- 180-220 bpm でのファストロール
- ペダルクローズのハイハット8分音符
- キックのみのオーディオインパルス
- 偽陽性ホールド: ゾーン内に休ませたスティック

生のビデオとオーディオは、寄稿者が明示的に共有に同意しない限り、ローカルに留まります。

## メトリクス

`scoreDrumBenchmark(expectedHitTimesMs, detectedHits)` は以下を報告します:

- 期待されるヒット数
- 検出されたヒット数
- 許容ウィンドウ内の一致したヒット
- 精度
- 再現率
- ゾーンごとの偽ダブルヒット数

ランナーは、一致したイベントの平均および p95 絶対タイミングエラー、ゾーン精度、手割り当て精度も記録します。

デフォルトのタイミング許容範囲は 35 ms です。ファストロールテストでは、ダブルトリガーの回帰を検出するために、ゾーンごとの最小分離が 35 ms である必要があります。

## パスゲート

- シングルヒット: 再現率 1.0, 精度 1.0
- 交互の手: 再現率 >= 0.95, 手割り当て >= 0.9
- ファストロール: 再現率 >= 0.9, 偽ダブルヒット = 0
- 偽陽性ホールド: 検出数 = 0
