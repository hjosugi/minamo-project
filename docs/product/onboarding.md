# First-Run Onboarding

The first screen should take users directly into setup:

1. Camera permission and privacy copy.
2. Camera, resolution, and FPS selection.
3. Lighting checklist with live quality chip.
4. Optional hands and upper-body toggles.
5. Neutral calibration capture.
6. Viewer launch link.
7. OBS setup link.

The onboarding state should be stored locally and resettable. It must never ask
users to upload camera or audio data.
