# ドラマーセットアップガイド

## カメラ

- スティック、手首、スネア、ハイハット、ライドが見えるようにカメラを高く設置します。
- 720p/60または1080p/60を使用します（利用可能な場合）。
- ステージのLEDからの強い逆光を避けます。

## キャリブレーション

1. トラッカーを起動します。
2. ドラマーモードを有効にします。トラッカーは自動的に手のトラッキングを有効にします。
3. ドラマーセットアップでゾーンを選択します。
4. **ゾーンを配置**を押し、そのゾーンが表示されるカメラプレビューをクリックします。
5. オーバーレイが演奏可能なエリアをカバーするまで半径を調整します。
6. スネア、ハイハット、ライド、クラッシュ、トム、キックについても繰り返します。
7. ドラムグリップを保持している間に手のデバッグとドラムゾーンリストが反応することを確認します。

## 信頼性チェック

- ファストロールはダブルトリガーなしで別々のヒットを生成するべきです。
- クロススティックやスティックオーバースティックの瞬間は左右の手を入れ替えないべきです。
- キックまたはハイハットペダルの推論が有効になっている場合、ライブで使用する前に音声と照合します。

## ドラマーモードセットアップ画面

トラッカーは次のものを持つ1つのセットアップサーフェスを公開します：

- カメラプレビューと手/スティックの信頼度
- ゾーンリスト：スネア、ハイハット、ライド、クラッシュ、トム、キック
- ゾーンごとの半径
- カメラプレビューからのクリックで配置するゾーンの中心
- オーディオオンセットメーター
- ベンチマーク結果の概要

ゾーンはローカル設定の `minamo.drum-kit.calibration.v1` に保持されます。
キャリブレーションデータはKGM1を通じて公開されず、ビューアオーバーレイはコンパクトな手のカールと手首のターゲットからライブ状態を導き出します。

## ペダル推論

ハイハットペダル：

- 予想されるティックの近くで高周波音のオンセットを検出します。
- 視覚的なハイハットゾーンの活動と組み合わせます。
- ヒットだけでなく、クローズ/オープンスカラーを出力します。

キックペダル：

- 低周波のオンセットを検出します。
- `zoneType: "kick"` を持つ `DrumHitEvent` を発信します。
- フットカメラモデルが有効でない限り、音声確認を必要とします。

## OBSオーバーレイ

トラッカーの **OBSオーバーレイURLをコピー** ボタンを使用するか、次を開きます：

```text
viewer/?preset=obs&room=stage&bg=transparent&hud=0&camera=locked&drum=1
```

現在のOBSフレンドリーなオーバーレイは、手から派生したドラムグリップの活動をレンダリングします：

- 透明な背景
- アクティブゾーンのハイライト付きゾーンラベル
- ジェスチャーと最寄りのゾーンを示す手ラベル
- カメラ動画なし

将来の `DrumHitEvent` の輸送は、OBS URLを変更することなく手から派生したパルスソースを置き換えることができます。
- カメラ動画なし
