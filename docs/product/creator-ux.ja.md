# Creator UX Design

> English version: [creator-ux.md](creator-ux.md)

## 1. First run

1. choose camera
2. check lighting
3. choose avatar type
4. quick calibration
5. open OBS setup guide

## 2. Advanced panel

- smoothing strength
- expression sensitivity
- finger sensitivity
- drum mode toggle
- privacy mode
- performance mode
- benchmark recording

## 3. Warnings

Warnings must be actionable.

Bad:

- `LOW_CONFIDENCE`

Good:

- `手が暗くて指先が見えません。ライトを正面に置くか、カメラ露出を上げてください。`

## 4. Presets

- low-end laptop
- normal webcam
- high-FPS webcam
- face-focused streamer
- hand-focused streamer
- drummer
- remote collaboration
