# 初回オンボーディング

最初の画面はユーザーを直接セットアップに導くべきです：

1. カメラの許可とプライバシーに関する文言。
2. カメラ、解像度、FPSの選択。
3. ライティングチェックリストとライブ品質チップ。
4. オプションの手と上半身のトグル。
5. ガイド付き顔キャリブレーション。
6. ビューワー起動リンク。
7. OBSセットアップリンク。

オンボーディングの状態はローカルに保存され、リセット可能であるべきです。ユーザーにカメラや音声データのアップロードを求めてはいけません。
