# ベンチマーキングと品質ゲート

## 1. ベンチマークが重要な理由

製品は長い機能リストではなく、自然な動きによって勝ちます。ベンチマークは目に見える安定性を測定する必要があります。

## 2. メトリクス

- エンドツーエンドのレイテンシ
- 推論レイテンシ
- FPS
- ドロップフレーム
- 顔のブレンドシェイプのジッター
- まばたきの誤検出
- 静止中の口のちらつき
- 指先のジッター
- 不可能な指のポーズのカウント
- 手の入れ替えのカウント
- ドラムヒットのタイミングエラー
- ドラムの誤検出/誤否定
- アバターの視覚的失敗のカウント

## 3. ゴールデンクリップ

小さな公開テストセットを作成します：

- 通常の照明
- 低照明
- 速い手の動き
- 隠れた手
- メガネ
- マスク
- ドラムスティック
- ドラムロール

## 4. 合格/不合格ゲート

P0ゲート：

- NaNフレームはレンダリングできない
- 警告なしにアバターに到達できる不可能な指のポーズはない
- 閾値未満の信頼度でのドラムヒットはない
- FPSターゲットはUIに表示される必要がある
- プライバシーモードは生のビデオを送信してはならない
