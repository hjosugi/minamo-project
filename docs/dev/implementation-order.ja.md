# 実装順序

## スプリント 1

- ウェブカメラを接続する
- デバッグオーバーレイをレンダリングする
- KGM1 フレームループを作成する
- FPS と信頼度パネルを追加する

## スプリント 2

- MediaPipe Hand Landmarker を統合する
- 指ごとのカール/スプレッドを導出する
- One Euro Filter を追加する
- 解剖学的クランプを追加する

## スプリント 3

- MediaPipe Face Landmarker を統合する
- まばたきを安定させる
- 口を安定させる
- Live2D/VRM パラメータにマッピングする

## スプリント 4

- ドラムキャリブレーション UI
- スティック検出器のプレースホルダー
- ビジュアルヒット検出
- オーディオオンセット検出器

## スプリント 5

- WebTransport プロトタイプ
- バイナリヘッダーコーデック
- リモートビューア

## スプリント 6

- ベンチマークハーネス
- ゴールデンクリップ
- CI チェック
- ドキュメントの仕上げ
