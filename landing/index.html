Local-first webcam avatar tracking

Fingers, eyes, mouth — even drums — moving naturally, from an ordinary webcam.

KGM1 is a tracking foundation for starting 2D/3D avatar streaming on a budget. It is designed around a stabilization layer that prevents jitter, bent joints, flips, and warping.

Quickstart

Clone it, open it on a local server. That's it.

git clone https://github.com/hjosugi/minamo-project.git
cd minamo-project
python3 -m http.server 8000
# open http://localhost:8000/landing/

Finger-perfect

Handles MCP/PIP/DIP/tip per finger, producing curl, spread, pinch, contact, velocity, and occlusion.

Eyes and mouth

Stabilizes blinks, gaze, iris, mouth opening, mouth width, lip rounding, smiles, and vowel-like mouth shapes.

Drum tracking

Integrates stick trajectories, drum-surface calibration, hit velocity, audio onsets, and foot pedals.

Anti-jitter

One Euro Filter, outlier rejection, velocity limits, skeletal constraints, and occlusion recovery prevent weird motion.

Issue-ready backlog

Ships with Markdown that's easy to file as GitHub issues later.

`issues/backlog/` contains 100+ issue candidates. Register them in bulk with `scripts/create_github_issues.py`.

python3 scripts/create_github_issues.py --repo OWNER/REPO --dry-run
python3 scripts/create_github_issues.py --repo OWNER/REPO --apply --label priority:P0
